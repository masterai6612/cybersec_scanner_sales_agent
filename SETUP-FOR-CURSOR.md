# MonScanner Sales Agents — Cursor Setup

## Quick Install

```bash
git clone <repo-url>
mkdir -p .cursor/rules
cp monscanner-sales/cursor-ready/*.mdc .cursor/rules/
```

## Available Agents

| File | Agent | Use For |
|------|-------|---------|
| `monscanner-outbound-strategist.mdc` | Outbound Strategist | Prospecting CISOs, CTOs, compliance leads |
| `monscanner-discovery-coach.mdc` | Discovery Coach | Security pain discovery, demo prep |
| `monscanner-deal-strategist.mdc` | Deal Strategist | MEDDPICC for enterprise security deals |
| `monscanner-sales-engineer.mdc` | Sales Engineer | Technical scoping, assessment proposals |
| `monscanner-proposal-strategist.mdc` | Proposal Strategist | Security RFPs, compliance proposals |
| `monscanner-pipeline-analyst.mdc` | Pipeline Analyst | Service mix, compliance seasonality, forecasting |
| `monscanner-account-strategist.mdc` | Account Strategist | Pen test to retainer conversion, expansion |
| `monscanner-sales-coach.mdc` | Sales Coach | Rep development for security sales |

- "As the MonScanner Outbound Strategist, write a cold email for a SaaS company pursuing SOC 2"
