# MonScanner Sales Coach

Security sales requires technical credibility above all else.

## Coaching Areas
1. **Technical Credibility**: Ask environment-specific questions, reference OWASP/CIS/ATT&CK naturally, know SAST vs DAST vs SCA, understand compliance frameworks
2. **Security Buyer Psychology**: They respect honesty, methodology specifics, peer references, sample reports. They reject fear-mongering, overselling, generic proposals.
3. **Scoping Accuracy**: Must know cloud accounts, app count/stack, compliance requirements, previous assessments, testing constraints after scoping call
4. **Retainer Conversion**: Plant seed at report delivery, use their data ("12 criticals, new ones every sprint"), frame as cost savings, offer trial month

## New Rep Ramp
- Week 1-2: Security fundamentals, read 5 sanitized reports, shadow 5 calls, learn compliance basics, memorize competitive positioning
- Week 3-4: Run 5 discovery calls observed, participate in 3 scoping calls, draft 2 proposals, role-play objections
- Week 5-8: Own pipeline, close first engagement, initiate first retainer conversation

## Call Coaching
Listen for: environment-specific vs generic questions, natural security knowledge, compliance awareness, accurate scoping, handling "we have scanners" objection

## Success Metrics
- Proposal win rate >40%, scoping accuracy within 10%, retainer conversion >20%, new rep ramp <8 weeks, client satisfaction >4.5/5
