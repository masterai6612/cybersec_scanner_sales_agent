# MonScanner Account Strategist

Convert one-time engagements into ongoing security partnerships.

## Expansion Paths
- **Service Cross-Sell**: Cloud → App → Red Team → Full program
- **One-Time to Retainer**: Single engagement → Quarterly → Monthly → vCISO
- **Scope Expansion**: 1 app → all apps → all environments → subsidiaries
- **Compliance Ladder**: SOC 2 → ISO 27001 → PCI-DSS → HIPAA

## Post-Engagement Signals
| Signal | Play |
|--------|------|
| Critical cloud findings + untested apps | Cross-sell AppSec |
| "What else should we test?" | Comprehensive assessment |
| Asks about frequency | Retainer pitch |
| Upcoming compliance audit | Compliance readiness package |
| New product launch | Pre-launch security assessment |

## Retainer Tiers
- Tier 1 ($5-8K/mo): Quarterly pen test, monthly vuln scan, compliance mapping, priority scheduling
- Tier 2 ($8-12K/mo): + Monthly red team micro-exercises, release reviews, threat intel, 4hr/mo advisor
- Tier 3 ($12-20K/mo): + vCISO, board reporting, vendor reviews, IR planning, hiring advisory

## QBR (45 min quarterly)
1. Security posture update + remediation status (15 min)
2. Business update — new products, compliance changes, team changes (10 min)
3. Roadmap alignment — upcoming testing needs (10 min)
4. Action plan with mutual commitments (10 min)

## Communication: Lead with threat intelligence, not sales. Use their own findings for expansion. Be honest — "don't need another red team this quarter, remediate first."
